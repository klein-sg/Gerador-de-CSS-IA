Gerador-de-CSS-IA

Um gerador de componentes front-end inteligente que transforma descrições em linguagem natural em código HTML/CSS funcional em segundos.

Tecnologias
- HTML & CSS
- JavaScript 
- Integração com IA

Funcionalidades
- Prompt to Code: Gere estruturas visuais apenas descrevendo-as.
- Live Preview: Veja o resultado renderizado em tempo real num iframe.
- Responsivo: Interface adaptada para dispositivos móveis.

- Nota: Para rodar este projeto localmente, 
é necessário obter uma chave gratuita na Groq Cloud 
e inseri-la na variável MINHA_CHAVE no arquivo script.js.
